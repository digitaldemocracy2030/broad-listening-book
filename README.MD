# 選挙を変えたブロードリスニング 生成AIが実現する民意の可視化と分析(仮)

[デジタル民主主義2030](https://dd2030.org/)の活動の一環として、広聴AI（ブロードリスニング）の技術とその社会実装の事例をまとめたものです。
