# 第6章 国政選挙でのブロードリスニングの利用

安野貴博氏の都知事選から1年。2025年の参議院選挙では、複数の政党がブロードリスニングを取り入れた。

安野氏は選挙期間中、X（旧Twitter）にこう投稿している[^anno-x-bl]。

> 2024年のブロードリスニング：
> 東京都知事選で謎の泡沫候補が唱えていた謎の概念
>
> 2025年のブロードリスニング：
> いろんな政党が採用し事実上の標準戦略に

一方で、各党が「ブロードリスニング」と呼んでいるものは、少しずつ異なっていたように思う。これは、どれかの方法が正しいということではない。それぞれの党が「ブロードリスニング」の概念を知ったことをきっかけに、人々とのよりよいコミュニケーション手法を模索している状態なのだと考えている。

各党はそれぞれ異なる制約の中で試行錯誤を重ねた。そこから得られた知見は、少しずつ制約の異なる地方選挙や民間企業での応用にも有用だろう。本章では、2025年参院選を中心に各党の取り組みを紹介する。

各党のアプローチを概観すると、収集チャネル・活用目的・実装規模はそれぞれ大きく異なる。

| 政党 | 主な収集チャネル | 活用目的 / 中心テーマ | 担当節 |
|---|---|---|---|
| チームみらい(2025参院選) | しゃべれるマニフェスト (AIチャット → GitHub) | マニフェストの市民共同編集 | 6.1 |
| チームみらい(2026衆院選) | 声が届くマニフェスト (チャット改善+受け止め強化) | 提案の取りこぼし削減・受信側設計の強化 | 6.2 |
| 日本維新の会 | 広聴AI / TTTC | 「社会保険料」を中心テーマとして掘り下げ | 6.3 |
| 国民民主党 | TTTC + こくみんAIプロジェクト + 電話・郵送 | 党員と市民の声を政策提言の根拠に | 6.4 |
| 公明党 | Google Form + LLM | 若年層の支持獲得策の立案 | 6.5 |
| 自由民主党(2026年4月始動) | じみんボイス: 党員専用ご意見フォーム + 議員から届く声 + AI分析[^jimin-voice] | 党員・党友の声を党運営と政策立案に反映 | 本書未掲載 |

ブロードリスニングは2025年参院選で「事実上の標準戦略」となり、2026年に入ってからは与党第一党の自民党までもが本格導入を表明するに至っている。

[^anno-x-bl]: 安野貴博 X投稿, 2025年5月14日. https://x.com/takahiroanno/status/1922524150150598818
[^jimin-voice]: 自由民主党「党員の声を広く聞く『じみんボイス』始動」2026年4月13日. https://www.jimin.jp/news/information/212986.html
